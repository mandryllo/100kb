# 100kb

Codebase for [100kb.space](https://100kb.space)\
Read more [here](https://100kb.space/about)

## Local Development

1. Install dependecies
```bash
pnpm install
```
2. Start the development server on `http://localhost:3000`
```bash
pnpm run dev
```
